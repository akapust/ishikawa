Lokal Datenschutz

Einleitung

Unsere Tools laufen lokal in deinem Browser. Je nach Nutzung gibt es zwei Datenschutz-Fälle: ohne KI/API und mit der optionalen Premium-KI/API-Funktion.

Kurzfassung

  1. Ohne KI/API: Keine Übertragung an unsere Server.
  2. Premium: Nur ausgewählte Inhalte gehen direkt an die konfigurierte OpenAI-API.
  3. Wichtig: Premium ist nur für Bestandskunden (Seminar/Inhouse, Beratung, Coaching).

Datenschutz (ohne API/KI) – werden Inhalte übertragen? +
Das Tool überträgt keine Inhalte an unsere Server. Der aktuelle Stand wird lokal im Browser gespeichert (Autosave/Browser-Speicher) und kann zusätzlich als JSON exportiert und wieder importiert werden.
Datenschutz (Premium mit API/KI) – was passiert bei Nutzung der KI? +
Die KI/API-Funktion ist eine Premium-Funktion und steht ausschließlich Bestandskunden zur Verfügung. Als Bestandskunde gilt, wer bereits ein Seminar (z. B. ein Inhouse-Seminar), eine Beratung oder ein Coaching absolviert hat.

Wenn du die Funktion nutzt, werden ausschließlich die von dir für die Anfrage ausgewählten Inhalte direkt aus deinem Browser an die konfigurierte OpenAI-API übertragen, um Vorschläge zu erzeugen. Der API-Key wird nicht serverseitig gespeichert. Bitte gib keine sensiblen personenbezogenen oder vertraulichen Daten ein.
Speichert ihr meinen API-Key oder Inhalte auf euren Servern? +
Nein. Ohne KI/API werden keine Inhalte an unsere Server übertragen. Bei Premium werden nur die von dir ausgewählten Inhalte direkt aus dem Browser an die konfigurierte OpenAI-API gesendet. Der API-Key wird nicht serverseitig gespeichert.
Kann ich meine Daten sichern? +
Ja. Der aktuelle Stand wird im Browser gespeichert (Autosave) und kann zusätzlich als JSON exportiert sowie später wieder importiert werden.
Was sollte ich aus Datenschutzgründen vermeiden? +
Bitte gib keine sensiblen personenbezogenen oder vertraulichen Daten ein – insbesondere nicht bei Nutzung der KI/API-Funktion.
Wie kann ich Premium erwerben? +
  1. Premium ist ausschließlich für Bestandskunden verfügbar.
  2. Bestandskunde ist, wer bereits ein Seminar (z. B. Inhouse), eine Beratung oder ein Coaching absolviert hat.
  3. Schreibe uns eine kurze Anfrage per E-Mail oder rufe an.
  4. Wir prüfen deinen Bestandskunden-Status und senden dir deinen individuellen Lizenzschlüssel.
  5. Lizenzschlüssel im Tool eintragen → Premium-KI/API-Funktion nutzen.
