Datenschutz & Nutzung

Häufige Fragen

lokal im Browser
1) Wo laufen die Tools? +
Die Tools laufen lokal in deinem Browser. Das bedeutet: Die Bedienung und Verarbeitung erfolgen direkt auf deinem Gerät.
2) Datenschutz (ohne API/KI) – werden Inhalte übertragen? +

Das Tool überträgt keine Inhalte an unsere Server. Der aktuelle Stand wird lokal im Browser gespeichert (Autosave/Browser-Speicher) und kann zusätzlich als JSON exportiert und wieder importiert werden.

3) Was genau wird lokal gespeichert? +
Gespeichert wird der aktuelle Arbeitsstand (z. B. Eingaben, Struktur, Zwischenergebnisse) im Browser-Speicher deines Geräts, damit du beim nächsten Öffnen weitermachen kannst.
4) Kann ich meine Daten exportieren und importieren? +
Ja. Du kannst deine Inhalte als JSON exportieren (Backup/Weitergabe) und später wieder importieren.
5) Was ist „Premium mit API/KI“? +
Die KI/API-Funktion ist eine Premium-Funktion und wird nur mit zusätzlichem Lizenzschlüssel freigeschaltet (jeder Kunde erhält einen individuellen Schlüssel).
6) Welche Daten werden bei Premium an die OpenAI-API übertragen? +
Wenn du die Funktion nutzt, werden ausschließlich die von dir für die Anfrage ausgewählten Inhalte direkt aus deinem Browser an die konfigurierte OpenAI-API übertragen, um Vorschläge zu erzeugen.
7) Wird mein API-Key serverseitig gespeichert? +
Nein. Der API-Key wird nicht serverseitig gespeichert.
8) Gibt es Inhalte, die ich nicht eingeben sollte? +
Bitte gib keine sensiblen personenbezogenen oder vertraulichen Daten ein – insbesondere nicht bei Nutzung der KI/API-Funktion.
9) Wie kann ich Premium erwerben? +
  1. Schreibe uns eine kurze Anfrage per E-Mail oder rufe an.
  2. Du erhältst Infos zu Lizenz/Abwicklung.
  3. Nach Freischaltung bekommst du deinen individuellen Lizenzschlüssel.
  4. Lizenzschlüssel im Tool eintragen → Premium-KI/API-Funktion nutzen.
